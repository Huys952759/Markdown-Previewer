# Front End Libraries Projects - Markdown Previewer

A simple markdown previwer

Here you can see the [preview](https://huys952759.github.io/Markdown-Previewer/).

## Installation

To get started right away:

1. install all project dependencies with ``` npm install ```
2. start the development server with ``` npm start ```
3. run ``` http://localhost:3000/ ``` to open the app

## User Stories

when entering GitHub flavored markdown into editors, the text is rendered as HTML in the previewer
